# Go 文件下载服务

## 简介
本项目是一个基于 Go 语言实现的简单文件下载服务。用户可以通过 HTTP 接口下载 `cyc` 目录下的任意文件。

## 使用方法
1. 启动服务：
   ```sh
   go run main.go
   ```
2. 下载文件：
   在浏览器或命令行中访问：
   ```sh
   http://<你的IP>:8080/download/<文件名>
   ```
   例如：
   ```sh
   wget http://192.168.10.198:8080/download/test.txt
   ```

## 日志
每当有文件被下载时，服务端会在控制台打印下载日志。
